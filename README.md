# Nigeria-Road-Analysis-Dashboard
Overview
This project is an interactive Power BI dashboard that analyzes road accident data in Nigeria. It explores accident patterns across states, weather conditions, vehicle types, road types, speed categories, and time to uncover meaningful insights that support road safety discussions.

Objectives
* Analyze accident trends and patterns.
* Identify factors associated with road accidents.
* Present insights through an interactive Power BI dashboard.

Tools
* Power BI Desktop
* Power Query
* DAX

Dataset
The dataset includes:
Accident ID
State
Date
Vehicle Type
Casualties
Weather
Road Type
Speed (km/h)

Dashboard Highlights
* KPI Cards (Total Accidents, Total Casualties, Average Speed)
Accidents by State
Accidents by Weather
Casualties by Road Type
Accidents by Vehicle Type
Monthly Accident Trend
Speed Category Analysis
Interactive Slicers

Key Insights
* Accident frequency varies across states and weather conditions.
* Road type and vehicle type influence casualty patterns.
* Higher speed categories are associated with a greater share of accidents.

Recommendations
* Strengthen road safety awareness and speed limit enforcement.
* Improve road safety measures during adverse weather conditions.
* Prioritize interventions in high-risk locations.
